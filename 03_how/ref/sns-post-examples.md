# SNS投稿例

## X（Twitter）用 - のぶのすけ向け

### 例1：休日ダラダラ系

```
休日15時。

まだパジャマ。
YouTube見てたら3時間経ってた。

「今日も終わった...」って思ってたら
過去の自分が「散歩したら意外と良かった」って言ってた。

10分だけ外出たら、なんか動けるようになった。

過去の自分、ありがとう。
```

### 例2：家事めんどい系

```
洗い物、3日放置してた。

シンク見るたびに「やらなきゃ...」って思うけど動けない。

過去の自分が「音楽かけたら10分で終わった」って書いてたから試した。

マジで10分で終わった。

過去の自分、天才か。
```

### 例3：夜ダラダラ系

```
夜23時。

「寝よう」と思ってから2時間スマホ触ってた。

過去の自分が「スマホ置いて本読んだら翌朝スッキリだった」って言ってた。

試してみるか...（続く）
```

### 例4：何もしたくない系

```
今日、マジで何もしたくない。

でも過去の自分が「そういう日はシャワーだけ浴びたら復活した」って書いてた。

試しにシャワー浴びたら、なんか動けるようになった。

過去の自分、よく知ってるな。
```

### 例5：朝起きれない系

```
朝9時。まだ布団の中。

「起きなきゃ」って思いながら1時間経過。

過去の自分が「カーテン開けたら起きれた」って書いてた。

腕だけ伸ばしてカーテン開けたら、本当に起きれた。
```

## Instagram/TikTok用 - みお向け

### 例1：動けない自己嫌悪系

```
また何もできなかった...

って思ってたら、
過去の自分が「そういう日もある」って言ってくれてた。

自分に優しくしていいんだ。
```

### 例2：日記続かない系

```
日記、書いても意味ない気がしてやめてた。

でも"書いて終わり"じゃなくて
"使える"日記があるらしい。

過去の自分が、今の自分を助けてくれる。
```

### 例3：休日昼まで寝る系

```
休日14時に起きて絶望してた。

でも過去の自分が
「昼から動いても間に合う」って教えてくれた。

そっか、まだ半日あるじゃん。
```

### 例4：仕事終わり何もできない系

```
仕事終わり、何もできない。

ソファで横になって、気づいたら寝落ち。

過去の自分が「帰ったらまず着替える」って書いてた。

それだけで、ちょっと動けるようになった。
```

## Instagram用 - ステップバイステップ系（カルーセル向き）

### 例1：朝起きれない → 外出まで

```
【朝起きれない時、過去の自分が教えてくれたステップ】

Step 1: カーテンを開ける
→ 腕だけ伸ばせばOK

Step 2: 布団から出る
→ 「3秒だけ」って思えば動ける

Step 3: 顔を洗う
→ 水の冷たさで目が覚める

Step 4: 好きな服に着替える
→ パジャマのままだと動けない

Step 5: コーヒーを淹れる
→ 香りで気分が上がる

Step 6: 外に出てみる
→ 朝の空気、意外と気持ちいい

過去の自分が「ここまでやったら1日動けた」って言ってた。
```

### 例2：家事やる気出ない → 片付けまで

```
【家事めんどい時、過去の自分が教えてくれたステップ】

Step 1: まず立ち上がる
→ 座ってると永遠に動けない

Step 2: 好きな音楽をかける
→ テンション上がる曲がベスト

Step 3: タイマーを15分セット
→ 「15分だけ」って思えば始められる

Step 4: 目の前の1つだけやる
→ シンクの皿1枚でOK

Step 5: 気づいたら終わってる
→ 始めたら意外と止まらない

過去の自分が「音楽かけたら15分で全部終わった」って言ってた。
```

### 例3：夜だらだら → 良い睡眠まで

```
【夜スマホやめられない時、過去の自分が教えてくれたステップ】

Step 1: 「あと5分」でスマホを置く
→ 宣言すると意外と置ける

Step 2: 部屋の明かりを暗くする
→ 眠くなってくる

Step 3: パジャマに着替える
→ 寝るモードに切り替わる

Step 4: ベッドに入る
→ スマホは遠くに置く

Step 5: 目を閉じる
→ 「3分だけ」って思えばいい

過去の自分が「これやったら翌朝スッキリだった」って言ってた。
```

### 例4：休日何もしたくない → 外出まで

```
【休日動けない時、過去の自分が教えてくれたステップ】

Step 1: とりあえずシャワーを浴びる
→ お湯で体が起きる

Step 2: ちゃんとした服に着替える
→ パジャマだと出かける気にならない

Step 3: 財布とスマホだけ持つ
→ 荷物多いと億劫になる

Step 4: 玄関を出る
→ 外に出たら勝ち

Step 5: 近くのカフェに行く
→ コーヒー片手に街を歩くと気分いい

過去の自分が「外出たら休日が充実した」って言ってた。
```

---

## 共通ポイント

- **短文で改行多め**（スマホで読みやすい）
- **「過去の自分」というフレーズを必ず入れる**（アプリの特徴）
- **最後はポジティブな変化で終わる**（希望を見せる）
- **自虐から入る**（共感を得やすい）
