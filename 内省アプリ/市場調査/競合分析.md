|----------|----------|----------|--------|------------------|------------|

## 競合分析まとめ

### ポジショニングマップ

```
                    AI活用度 高
                        ↑
                        │
         Wysa ●─────────┼─────────● Reflection.app
                        │
                        │    ● Reflectly
                        │
  シンプル ←────────────┼────────────→ 多機能
                        │
              リフクラ ● │
                        │
            Jour ●──────┼───────● Day One
                        │
        Mind Journal ●  │
                        │
                        ↓
                    AI活用度 低
```

### 差別化の機会

1. **AI × 日本語**: AI活用した日本語ネイティブアプリが少ない
2. **コーチング型**: 単なる記録ではなく、成長を促すフィードバック
3. **コミュニティ**: 孤独な内省ではなく、仲間との繋がり
4. **データ可視化**: 感情・成長の長期トレンド分析
