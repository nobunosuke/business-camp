# Folder Structure Ideas

## Plan 1: Story-Driven

```
/01_why/                        # なぜやるか
    problem.md                  # ターゲットの悩み、課題
    market.md                   # 市場規模、タイミングの根拠

/02_what/                       # 何をやるか
    concept.md                  # サービスの核、ターゲット、提供価値
    mockup/                     # 画面イメージ

/03_how/                        # どうやるか
    competitive-advantage.md    # なぜ勝てるか
    sales-strategy.md           # どう売る、どう集客する
    revenue-and-kpi.md          # いくら稼ぐ、何を測る
    roadmap.md                  # いつ何やる

/04_presentation/               # 発表資料

/references/                    # 参考資料（競合スクショ、調査PDFなど）
```

## Plan 2: Minimal ⭐Recommended

```
/product/                       # プロダクト
    concept.md                  # サービス全体像、ターゲット、課題、解決策
    mockup/                     # 画面イメージ

/strategy/                      # 事業戦略
    market-and-competitors.md   # 市場規模 + 競合分析 + 優位性
    revenue-and-plan.md         # 収益シミュレーション + KPI + ロードマップ + 販売戦略

/presentation/                  # 発表資料
```

## Plan 3: Output-Oriented

```
/product/                       # プロダクト（作るもの）
    concept.md                  # コンセプト
    mockup/                     # モックアップ

/business/                      # ビジネス（売り方）
    market-analysis.md          # 市場分析
    competitive-advantage.md    # 競合優位性
    revenue-model.md            # 収益モデル
    roadmap.md                  # ロードマップ

/presentation/                  # プレゼン
    story-outline.md            # プレゼンの台本・流れ

/_assets/                       # 素材（調査PDF、競合スクショなど）
```
