# LLM API 価格比較メモ（2025年11月時点）

## 比較結果サマリー

**決定: GPT-5 nano を採用**

理由:
- 月額コスト/ユーザーが最安（¥20/月）
- 黒字化に必要なMAUが最小（5万人）
- 内省アプリの用途（要約・分類）に適している

## 結論

GPT-5 nanoとGPT-5 miniで**5倍のコスト差**がある。

内省アプリの主な用途（ジャーナル整理、おすすめ生成）は「要約・分類」に近いタスクなので、GPT-5 nanoで十分な品質が出せる可能性が高い。

もし品質に問題があれば、後からGPT-5 miniにアップグレードすればよい。
